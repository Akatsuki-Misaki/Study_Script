# 说明

  这是一个自动化音频转换格式的脚本
  因为wav格式不支持内部歌词数据所以要转为flac格式
  运行前请先安装依赖库

```
pip install pydub mutagen eyed3 
pyinstaller -i ./q.ico --onefile --version-file version_info.rc main.py
```
